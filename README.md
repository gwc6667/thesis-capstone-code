This repository contains the source code, thesis materials, dataset notes, and figure source files for my capstone thesis project.

## Project Title
**A Unified Artificial Intelligence Framework for Academic Performance Prediction and Emotion Analysis**

## Main Components of the Project

### Academic Prediction Branch
This part of the project focuses on predicting academic performance using sequential academic and behavioral data.  
It includes:
- data preprocessing
- feature extraction
- temporal modeling
- training and evaluation

### Emotion Analysis Branch
This part focuses on emotion-related representation learning using image-based or video-based affective data.  
It includes:
- face-related preprocessing
- emotion representation extraction
- model training and evaluation

### Multimodal Fusion
This part combines academic and emotion-related representations to improve prediction performance and robustness.  
Fusion strategies considered in this project include:
- feature-level fusion
- gated fusion
- attention fusion

## Datasets

### Academic Dataset
**OULAD (Open University Learning Analytics Dataset)**  
Used for academic performance prediction and behavioral analysis.

### Emotion Dataset
**eNTERFACE'05**  
Used for emotion-related visual input and affective representation learning.
